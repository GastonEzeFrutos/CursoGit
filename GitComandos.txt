GIT Comandos:

1- git init: Dar comienzo.
2- git status -s: Estado de los documentos.
3- git add + nombre del archivo: lo envia al área de Ensayo.
4- git commit -m + "descripción de lo que queramos".
5- git log --oneline: listado de commits.
6- git reset -hard + codigo del commit al que queremos volver.
7- git push: sube los cambios a github.
8- git pull: trae los cambios al documento en que se este trabajando.

tags: permite establecer versiones a nuestro proyecto.

9- git tag 27-12-23v1 -m "Version 1 del proyecto"
10- git push --tags
11- git clone : para pegar la url que copiamos en el portapaples.
